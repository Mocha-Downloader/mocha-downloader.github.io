# Contributing

## Setting up

### 0. Install pre-requirements.

- git
- Node.js & npm
- yarn

### 1. Clone the repository.

```
git clone https://github.com/Mocha-Downloader/mocha-downloader.github.io.git
```

### 2. Install npm dependencies.

```
yarn install
```

### 3. Test locally.

```
yarn start
```

Test different locale:

```
yarn start --locale <code>
```

available locale:

- en
- ko
